# Assignment 25 :
This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept N number from user & return the difference between summation of even elements & summation of odd elements.
2. Accept N number from user & display all such elements which are divisible by 5.
3. Accept N number from user & display all such elements which are even and divisible by 5.
4. Accept N number from user & display all such elements which are divisible by 3 and 5.
5. Accept N number from user & display all such elements which are multiples of 11.
