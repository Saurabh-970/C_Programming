Assignment 22 :
This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept N number from user & return Frequence of Even number.
2. Accept N number from user & return diff Frequence of Even & Odd num.
3. Accept N number from user & Check number contains 11 in it or not.
4. Accept N number from user & return diff Frequence of 11 from it.
5. Accept N number from user & accept one another number as NO return frequence of NO.
Commands used to execute these codes are:-

gcc filename.c -o myexe

myexe
