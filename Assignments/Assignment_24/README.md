# Assignment 24 :
This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept N number from user & return the Largest number.
2. Accept N number from user & return the Smallest number.
3.Accept N number from user & return the difference between largest and smallest number.
4.Accept N number from user & display all such number which contains 3 digits in it.
5.Accept N number from user & display summation of ditits of each number.
Commands used to execute these codes are:-

gcc filename.c -o myexe myexe
