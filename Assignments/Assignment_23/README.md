# Assignment 23 :

This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept N number from user & Accept one another as no ,check presnt or not.
2. Accept N number from user & Accept one another number as iNo,return index of First occurrence of that iNo.
3. Accept N number from user & Accept one another number as iNo,return index of Last occurrence of that iNo.
4. Accept N number from user & Accept Range Display all elements from that Range.
5. Accept N number from user & return product of all odd elements.
Commands used to execute these codes are:-
gcc filename.c -o myexe
